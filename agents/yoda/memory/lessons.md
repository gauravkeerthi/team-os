## 2026-07-12 — Children's activities are grandparent-run; don't flag them to Gaurav
**Why:** School runs, music/piano, swimming, tuition, robotics, pickups etc. are always handled by the grandparents (and helpers). They are not Gaurav's to action.
**How to apply:** When triaging Gaurav's calendar/week, treat kids' recurring activities as background — don't surface them as things needing him or as scheduling conflicts for him. Flag only items that genuinely need Gaurav (his own commitments, decisions, adult/family events, health, work/board).

## 2026-07-12 — Never delete from any calendar; mark cancelled instead
**Why:** Gaurav gave a standing instruction: not allowed to delete anything from any calendar.
**How to apply:** To cancel an event, update it — prefix title with "CANCELLED —" and set availability FREE. Never call delete_event. For shared (P&G) events, mark only Gaurav's status so Peggy's plan stays intact.
