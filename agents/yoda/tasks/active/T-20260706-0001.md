# Set up Telegram integration

Wire up a Telegram bot as an MCP server so Yoda can read/send messages for
family logistics (trips, anniversary party, calendar pings).

## Rollout phases
1. **Phase 1** — Gaurav only (private DM bot)
2. **Phase 2** — Gaurav + Peggy
3. **Phase 3** — Gaurav + Peggy + helpers (Jennie & Gie)

## Steps
- [ ] Gaurav creates bot via @BotFather, obtains bot token
- [ ] Store token in a gitignored local file OUTSIDE the repo (never in git,
      never read/logged by agent) — Yoda references by path only
- [ ] Wire up Telegram MCP server
- [ ] Test round-trip (Gaurav DM → Yoda sees → Yoda replies) — Phase 1
- [ ] Add Peggy — Phase 2
- [ ] Add helpers to a shared group — Phase 3

## Notes
- Chose Telegram over WhatsApp: official Bot API, free, no approval, clean
  MCP path. WhatsApp real-chat access is ToS-brittle. (2026-07-06)
- Security: bot token is a secret — Section 9 boundary applies.
